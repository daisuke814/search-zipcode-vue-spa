# 郵便番号から住所を検索するVue SPA

郵便番号を入力すると住所を検索し表示します。  
郵便番号はハイフンありでも検索可能です。  
郵便番号の検索は、[zipcloud](http://zipcloud.ibsnet.co.jp/doc/api)のAPIを使用しています。

## 使用ライブラリ・コンポーネント

* Vue 3
* Vuetify 3 (Beta)
* Vue Router
* axios
* Vue Loading Spinner
